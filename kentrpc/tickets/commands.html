Welcome to KentRPC Ticket System
Here you can find all the commands you need to manage tickets efficiently.
Tickets by Thompson
Ticket Commands
/add
People Services
This command will add a user that you specify to the ticket in which you execute the command in.
/add-role
People Services Supervisor
This command will add a role that you specify to the ticket in which you execute the command in.
/close
People Services
This command will close a ticket.
/remove
People Services
This command will remove a user that you specify from the ticket in which you execute the command in.
/remove-role
People Services Supervisor
This command will remove a role that you specify from the ticket in which you execute the command in.
/rename
People Services
This command will rename the ticket channel to whatever you specify.
SLT Ticket Commands
/panels
SLT
This command will allow you to toggle the functionality of each ticket panel.
/tickets-setup
SLT
This command will setup the ticket panel in the channel you execute the command in.
/interviews-setup
SLT
This command will setup the interviews enquiry panel in the channel you execute the command in.
